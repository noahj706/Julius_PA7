# Attendance Tracker With Classes | Julius_PA7

An assignment from my data structures course, uses C++ classses and templates to enable a user to do the following:
- Import class list
- Load previous master class list, this include absence info per student
- Store current master class list
- Mark absences
- Edit absences (BONUS feature)
- Generate reports on students' last absence and students with x or more absences

Annnnd thats it. Any strange design choices were due to assignment instructions I swear.
